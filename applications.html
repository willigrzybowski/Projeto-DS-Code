Visual Studio Code

Um editor leve e altamente personalizável, com suporte a uma ampla gama de linguagens e extensões. É uma ótima opção para iniciantes e desenvolvedores experientes.

Visual Studio 2022

Codifique com rapidez e precisão, usando sugestões de código do IntelliSense. Conclua linhas inteiras de código com sugestões do IntelliCode habilitado para aprendizado de máquina.

Apache NetBeans

Um IDE de código aberto que suporta várias linguagens de programação, incluindo Java, PHP, e HTML5. Oferece recursos como edição de código, depuração, e gerenciamento de projetos, sendo uma opção popular para desenvolvimento Java.

XAMPP

Um pacote de software gratuito e de código aberto que inclui Apache (servidor web), MySQL (banco de dados), PHP, e Perl. É usado para criar um ambiente de desenvolvimento local para testar e desenvolver aplicações web.

Node.JS

Um ambiente de execução JavaScript do lado do servidor baseado no motor V8 do Google Chrome. Permite criar aplicações web escaláveis e rápidas usando JavaScript no lado do servidor, além de ter um grande ecossistema de pacotes disponíveis via npm (Node Package Manager).

React Native

É uma ferramenta que permite criar aplicativos para iOS e Android usando JavaScript. Em vez de escrever códigos separados para cada plataforma, você usa o mesmo código para ambos, tornando o desenvolvimento mais rápido e eficiente.

Usb WebServer

É um software que combina um servidor web (Apache), banco de dados (MySQL) e suporte para PHP, tudo em um único pacote. Ele é projetado para ser executado a partir de uma unidade USB, facilitando o desenvolvimento e teste de sites e aplicativos web de forma portátil.

Figma

É uma ferramenta online para design de interfaces e protótipos. Permite que várias pessoas trabalhem juntas no mesmo projeto em tempo real, facilitando a colaboração e a criação de designs para aplicativos e sites.

Adobe Premiere

Aplicativo de edição de vídeo profissional. Usado para cortar, montar e criar vídeos com uma ampla gama de ferramentas e efeitos, porém é pago.

Adobe Ilustrator

Ferramenta de design gráfico baseada em vetores. Ideal para criar ilustrações, logotipos e gráficos que podem ser escalados sem perda de qualidade, porém é pago.

Adobe Photoshop

Software líder em edição de imagens e design gráfico. Permite retocar, manipular e criar imagens digitais com uma vasta gama de ferramentas e filtros, porém é pago.

DaVinci Resolve

DaVinci Resolve é um aplicativo de correção de cores e edição de vídeo não-linear para macOS, Windows e Linux, parecido com o Adobe Premiere, mas distribuído de maneira gratuita.

GIMP

GIMP é um editor de imagens multiplataforma disponível para Windows, macOS, Linux e mais sistemas operacionais. É software livre com código aberto que você pode mudar e distribuir, parecido com o Adobe Ilustrator e com o Adobe Photoshop, mas distribuído de maneira gratuita.

7-Zip

É um programa de compressão de arquivos que reduz o tamanho dos arquivos e pastas para facilitar o armazenamento e a transferência. Suporta vários formatos de compressão, como .zip e .7z.

WinRAR

WinRAR é um software compactador e descompactador de dados, distribuído pela licença shareware

Android Studios

É um ambiente de desenvolvimento integrado (IDE) oficial para criar aplicativos Android. Oferece ferramentas para codificação, depuração, e design de interfaces de usuário. Cuidado, ele é muito pesado.

Git

Git é um sistema de controle de versões distribuído, usado principalmente no desenvolvimento de software, mas pode ser usado para registrar o histórico de edições de qualquer tipo.

GitHub

É uma plataforma baseada na web que usa Git para hospedagem de repositórios de código. Facilita a colaboração, compartilhamento e revisão de código entre desenvolvedores.
